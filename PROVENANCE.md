# Provenance

This repository is a curated public export of local operator skills.

Included skills were selected because they are generic enough to be useful outside a private workspace. Private product, account, vault, outreach, analytics, screen-history, and vendor/plugin skills were excluded.

The public copies were sanitized before publication. Some skills were rewritten from private operating patterns into generic templates. The repository does not bundle third-party skills that are already distributed by their original maintainers.

This project is not affiliated with or endorsed by OpenAI, Anthropic, Vercel, GitHub, or any other platform named in examples.

## Initial Included Set

- `codex-improvement-loop`: rewritten into a generic multi-step coding orchestration pattern.
- `honest-output-auditor`: rewritten into a generic hard-audit workflow.
- `seo-audit`: script defaults generalized and made configurable.
- `changelog-generator`: included as a generic changelog workflow.
- `cli-creator`: included as a generic CLI creation workflow.
- `define-goal`: included as a generic goal-definition workflow.
- `security-best-practices`: included as a generic framework-specific secure-coding reference pack.
- `security-threat-model`: included as a generic repository-grounded AppSec workflow.
- `security-ownership-map`: included as a generic git-history security ownership analysis workflow.
